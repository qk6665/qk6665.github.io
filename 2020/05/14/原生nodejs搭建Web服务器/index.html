1 创建 Web 服务器

  1. 创建服务器app = http.createServer()
  2. 监听请求app.on('request', (req, res) => {}，req 为接收的请求信息，res 为发送的响应信息
  3. 监听端口app.listen(3000)
  4. 解析 urlurl.parse(req.url, true)
  5. 路由转发
